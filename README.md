# JanBroz – Artist Profile Website

A dynamic React website for JanBroz, twin dancers from Hunza, blending tradition and modernity through dance. Built with Vite for speed and modern development.

## Features
- **About/Our Story:** Learn about the journey and inspiration behind JanBroz.
- **Performance Highlights Dashboard:** Dynamic, easy-to-update list of major performances and achievements.
- **Responsive Design:** Looks great on all devices.
- **Modern Stack:** React, Vite, and React Router for fast navigation.

## Getting Started

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Start the development server:**
   ```sh
   npm run dev
   ```
3. **Open your browser:**
   Visit [http://localhost:5173](http://localhost:5173) to view the site.

## Project Structure
- `src/Navbar.jsx` – Navigation bar
- `src/Home.jsx` – Home/Intro section
- `src/About.jsx` – Story/About section
- `src/Dashboard.jsx` – Performance highlights dashboard
- `src/App.jsx` – Main app with routing

## Customizing Highlights
To update the performance highlights, edit the `highlights` array in `src/Dashboard.jsx`.
